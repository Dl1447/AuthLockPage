# AuthLock 宣传网页

## 项目简介

AuthLock 宣传网页是为 AuthLock 版权隐形指纹保护工具设计的官方网站。该网页使用现代前端技术构建，展示了 AuthLock 的功能特性、技术栈、下载链接和支持信息。

## 页面结构

- **导航栏**：包含功能特性、技术栈、下载和支持等导航链接
- **英雄区域**：展示 AuthLock 的核心价值主张
- **功能特性**：详细介绍 AuthLock 的主要功能
- **技术栈**：展示 AuthLock 使用的技术
- **下载区域**：提供 Windows 版本的下载链接
- **支持区域**：包含爱发电支持信息
- **隐私政策**：详细的隐私保护说明
- **使用条款**：软件使用的法律条款
- **页脚**：包含版权信息、官网链接和联系方式

## 技术栈

- **HTML5**：页面结构
- **CSS3**：页面样式
- **Tailwind CSS**：实用优先的 CSS 框架
- **Font Awesome**：图标库
- **JavaScript**：交互功能

## 如何使用

1. 将 `page` 文件夹部署到任何静态网站托管服务
2. 确保 `ui.png` 和其他资源文件正确引用
3. 访问部署后的网站即可查看 AuthLock 宣传页面

## 本地预览

1. 导航到 `page` 文件夹
2. 使用本地服务器打开 `index.html` 文件
   - 例如：`python -m http.server 8000`
   - 然后在浏览器中访问 `http://localhost:8000`

## 自定义

- 修改 `index.html` 文件可以更改页面内容
- 修改 `<style>` 标签内的 CSS 可以调整页面样式
- 替换 `ui.png` 可以更新界面截图
- 更新链接可以指向不同的资源
